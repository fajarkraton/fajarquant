## Evaluation Protocol (R-alpha.1)

- Dataset: WikiText-2-raw-v1 test split (Salesforce/wikitext)
- Chunk length: 2048 tokens (non-overlapping)
- Max samples: 30 chunks (61,410 tokens total)
- Method: Model surgery — patch attention forward pass to quantize KV in-flight
- FP16 baseline: Same forward pass with identity (no quantization) patch
- Calibration: Per-model `.npz` files in `data/calibration/`

## Reproducing All Results

```bash
# 1. Clone and set up
git clone https://github.com/fajarkraton/fajarquant
cd fajarquant

# 2. Python environment
python3 -m venv .venv
source .venv/bin/activate
pip install torch transformers datasets numpy

# 3. Run canonical perplexity evaluation (requires GPU)
# Gemma (~30 min)
python3 scripts/eval_perplexity_v2.py \
  --model google/gemma-4-E2B \
  --calibration data/calibration/fq_v2_gemma_4_e2b.npz \
  --seq-len 2048 --max-samples 30 --bits 2,3,4 \
  --output data/kv_cache/perplexity_v2_gemma.json

# Mistral (~1.5h)
python3 scripts/eval_perplexity_v2.py \
  --model mistralai/Mistral-7B-v0.1 \
  --calibration data/calibration/fq_v2_mistral_7b_v0.1.npz \
  --seq-len 2048 --max-samples 30 --bits 2,3,4 \
  --output data/kv_cache/perplexity_v2_mistral.json

# Qwen2-7B (~1.5h)
python3 scripts/eval_perplexity_v2.py \
  --model Qwen/Qwen2-7B \
  --calibration data/calibration/fq_v2_qwen2_7b.npz \
  --seq-len 2048 --max-samples 30 --bits 2,3,4 \
  --output data/kv_cache/perplexity_v2_qwen2.json

# 4. Verify paper claims against data
python3 scripts/verify_paper_tables.py --strict
# Expected: "OK — all 28 claims verified within tolerance"
```

## Random Seeds

PyTorch RNG seeding is not fixed for evaluation (perplexity is
deterministic given the model weights and input text). Slight
cross-run variation in relative error of synthetic self-tests
(±0.5% at 2-bit, ±0.2% at 4-bit) is expected due to process-level
RNG differences and does not affect reported PPL numbers.

## Calibration Data

Calibration `.npz` files in `data/calibration/` were generated by
`scripts/calibrate_v2.py` (committed in B3.2) using 128 WikiText-2
training samples per model. The calibration is deterministic given
fixed model weights and input text.
